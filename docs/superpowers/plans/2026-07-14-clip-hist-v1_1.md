# clip-hist v1.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add pins, an inline preview pane, an ignored-apps setting, secret detection, copy notifications, and open-source hygiene (copy-install, committed tests, CI) to clip-hist.

**Architecture:** Unchanged watcher/store/CLI/widget split. Pins are a second JSONL store merged into the read path ahead of history. Ignore/secrets are watcher-side filters driven by `config.json`. All settings surfaces extend the existing Ctrl+S fzf menu.

**Tech Stack:** Same as v1 (Swift/AppKit, bash 3.2, /usr/bin/python3, zsh/fzf, launchd) + GitHub Actions (macOS runner), shellcheck.

**Spec:** `docs/superpowers/specs/2026-07-14-clip-hist-design.md` — section "v1.1 additions".

## Global Constraints

- Repo `the repo root`, branch per task series `v1.1`. Bash 3.2 only. No emojis anywhere.
- Stores: `history.jsonl` (retention-pruned) and NEW `pins.jsonl` (never pruned, max **100** pins, mode 600) in `$DATA_DIR` (`CLIPHIST_DATA_DIR` override).
- Combined read order everywhere (list/get/pick-feed/pin): pins newest-first, then history newest-first; one shared index space.
- Config keys: `retention`, `key`, NEW `ignored_apps` (array of bundle ids), NEW `skip_secrets` (`"on"` default / `"off"`).
- **From this version tests are COMMITTED** (explicit exception to the session-only rule): `tests/` leaves .gitignore in Task 6 and all test scripts are added to git there. Until Task 6, keep committing only source files.
- The watcher tests and `test_misc.sh`/`test_watcher_privacy.sh` touch the LIVE clipboard (saved/restored); `test_watcher_app.sh` and `test_watcher_privacy.sh` briefly steal focus (Finder) and restore it. Never run them in CI.
- Commits: plain messages, no co-author or attribution lines.

```bash
hist, pins, cfg_path = sys.argv[1], sys.argv[2], sys.argv[3]
seen = set()
for path in (hist, pins):
    try:
        for line in open(path):

except Exception:
    pass

for a in sorted(seen | installed | ignored, key=str.lower):
    print("[%s] %s" % ("x" if a in ignored else " ", a))
```

```swift
func shannonEntropy(_ s: String) -> Double {
    var freq: [Character: Double] = [:]
    for c in s { freq[c, default: 0] += 1 }
    let len = Double(s.count)
    guard len > 0 else { return 0 }
    return freq.values.reduce(0) { $0 - ($1 / len) * log2($1 / len) }
}

func looksLikeSecret(_ text: String) -> Bool {
    let s = text.trimmingCharacters(in: .whitespacesAndNewlines)
    if s.contains("PRIVATE KEY-----") { return true }
    guard !s.contains(where: { $0.isWhitespace || $0.isNewline }) else { return false }
    let prefixes = ["ghp_", "github_pat_", "gho_", "sk-", "AKIA", "xoxb-", "xoxp-", "AIza"]
    if s.count >= 20, prefixes.contains(where: { s.hasPrefix($0) }) { return true }
    if s.hasPrefix("eyJ"), s.contains("."), s.count > 60 { return true }
    if s.count >= 32, s.count <= 256, !s.contains("://"), !s.hasPrefix("/") {
        let classes = [s.contains(where: { $0.isUppercase }),
                       s.contains(where: { $0.isLowercase }),
                       s.contains(where: { $0.isNumber })].filter { $0 }.count
        if classes >= 3, shannonEntropy(s) >= 4.0 { return true }
    }
    return false
}
```

```bash
cmd_copy() {
  # Sanctioned agent write path: copy TEXT (or stdin) to the clipboard.
  local quiet=0 content snippet
  if [[ "${1:-}" == "--quiet" ]]; then quiet=1; shift; fi
  if [[ $# -gt 0 ]]; then content="$*"; else content=$(cat); fi
  printf '%s' "$content" | pbcopy
  if [[ $quiet -eq 0 ]]; then

  fi
  echo "copied to clipboard"
}
```
